# Lab Build Plan

## Objective
Build a virtual enterprise-style IT environment for help desk and system administration practice.

## Planned Infrastructure
- 1 Windows Server Domain Controller
- 2 Windows Client Machines
- Active Directory Domain
- DNS
- DHCP
- Shared Folders
- Group Policy

## Planned Ticket Scenarios
- Password resets
- Account lockouts
- Shared drive access issues
- DNS failures
- DHCP issues
- Remote desktop troubleshooting
