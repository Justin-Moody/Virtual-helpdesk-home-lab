# Ticket Documentation

This folder contains simulated help desk tickets used for troubleshooting practice.

## Planned Ticket Types
- Password resets
- Account lockouts
- Printer issues
- DNS problems
- DHCP failures
- Shared drive permissions
- VPN troubleshooting
- Outlook issues
- Remote desktop support

Each ticket will include:
- Problem description
- Troubleshooting steps
- Resolution
- Lessons learned
